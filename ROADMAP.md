# つんどけんさく ロードマップ

最終更新: 2026-07-07

## コンセプト

- 積読を知識に変える
- AI（NotebookLM や ChatGPT など）へ渡す資料を組み立てる、ローカル完結型ワークスペース

つんどけんさくは AI 要約・ベクトル検索・RAG を内蔵しません（[README「あえて入れていないもの」](README.md#あえて入れていないもの) 参照）。AI はあくまで外部のサービスとして使い、このアプリは **AI に渡す前（資料の組み立て）と、AI から戻った後（知識の定着）** を支えることに徹します。

## 現在地

### Phase 0: 検索基盤 — 完了

「蔵書から探す」は一段落しました。

- 蔵書横断検索: PDF 本文・タイトル・Kindle 本・Scrapbox/Cosense メモ・書籍ノート（FTS5 + Sudachi 分かち書き + trigram 部分一致）
- 検索条件: AND/OR 切替（「すべての語を含む」チェックボックス）、`"フレーズ検索"`、`-除外語` / `-"フレーズ除外"`
- タイトル推定: PDF メタデータ → Scrapbox → ファイル名整形のフォールバック、OCR ノイズ除外、`refresh-titles` コマンド
- 資料組み立ての原型: ワークスペース（検索結果・本文検索モーダル・PDF プレビュー・アウトライン章選択からのページ追加）、PDF ページ切り出し、Markdown エクスポート

### Phase 1: パック化 — MVP 完了（2026-07-07）

ワークスペースを「問い（テーマ）」単位の **名前付きパック** として SQLite に永続化しました。かつての sessionStorage 揮発カートは廃止し、初回アクセス時に自動でパックへ移行されます。

実装済み:

- パックの作成・切替・改名・削除（ワークスペースのセレクトボックス）
- 検索結果・PDF プレビューからの追加はアクティブパックへ保存され、ブラウザを閉じても残る
- 出典情報の基礎（追加時点のタイトル・ページ範囲・追加日時）を保存 — Phase 2 以降（エクスポートプロファイル・AI 成果物の帰還・消化マップ）のデータ基盤

## ロードマップ

### Phase 2: 宛先別エクスポートプロファイル + トークンバジェット

- NotebookLM 向け（PDF、ソース数上限を考慮）、ChatGPT/Claude 向け（Markdown、コンテキスト量を考慮）など、宛先ごとの整形・分割・命名規則をプロファイル化
- エクスポート前に概算トークン数・ページ数を表示し、「渡しすぎ / 足りない」を見えるようにする

既存のエクスポート機能の延長にあり、実装距離が最も短く即効性が高い段階です。

### Phase 3: ラウンドトリップ（AI 成果物の帰還）

- AI から得た回答・ノートを取り込み、生成元のパック（どの本のどのページから作った資料か）に紐づけて保存
- 帰還した知識は蔵書横断検索の対象になる（メモと同格の新しいソース）

現状では AI とのやりとりで得た知見が AI サービス側に置き去りになります。この段階で「積読 → 資料 → AI → 知識」のループが初めて閉じ、コンセプトの「知識に変える」が完結します。

### Phase 4: 消化マップ

- パックの利用実績から、本ごとの「消化率」（どのページを切り出し・エクスポート・パック追加したか）を可視化
- 「この積読は実は8割解体済み」「この本は3章しか使っていない」が見える

読書管理アプリの読了フラグと異なり、実際の利用実績ベースであることがつんどけんさくらしさです。Phase 1 のデータが溜まるほど効きます。

### Phase 5: 対比パック / 共起提示

- 対比パック: 同じ問いに対する複数書籍の記述を並べて1つの資料にする。AI に比較・統合させるための資料構成
- 共起提示: パックで一緒に使われたページ同士の関係から「このページと一緒によく使われるページ」を提示する。ベクトル検索を使わず、自分の利用履歴だけによるローカルな関連付け

## 優先順位の要約

1. ~~**パック永続化**~~ — MVP 完了（2026-07-07）
2. **エクスポートプロファイル** — 最短距離で「AI に渡す」体験を向上。次の最優先
3. **ラウンドトリップ** — 「知識に変える」ループの完結
4. **消化マップ** → 5. **対比・共起** — 利用履歴の蓄積が効く順

## やらないこと

- AI 要約・ベクトル検索・RAG の内蔵（AI は外部サービスに任せる）
- 一般的な PDF ビューア・ファイル管理ソフトの機能の追いかけ
